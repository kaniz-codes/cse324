## What is Compiler?
A compiler is a system software that translates a program written in a high-level programming language into low-level code, such as assembly language or machine code.

### Compiler Design is used for

Compiler Design concepts are used to understand and build language-processing systems such as:

- Compilers
- Interpreters
- Assemblers
- Linkers
- Loaders
- Syntax checkers
- Code analyzers
- Programming language tools
- Code optimization tools
